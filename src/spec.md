# Specification

## Summary
**Goal:** Create a responsive, classic single-page marketing homepage for a Coimbatore (Tamil Nadu, India) steam ironing, laundry, and dry cleaning business, using a professional layout with free stock-style imagery.

**Planned changes:**
- Build a single-page layout with sections: Hero, Services, Why Choose Us, How It Works, Testimonials, Gallery, Contact.
- Add header navigation with smooth-scroll links to each section.
- Apply a consistent classic visual theme (typography + warm/neutral palette) avoiding blue/purple as the primary theme.
- Implement Hero copy with prominent “Call” (tel:) and “WhatsApp” (wa.me) CTA buttons.
- Add Services cards for: Steam Ironing, Dry Cleaning, Stain Removal, Darning, Saree Pleating & Ironing (each with short descriptions).
- Add “Why Choose Us” (4–6 trust highlights) and “How It Works” (3–4 steps with non-committal pickup/delivery wording).
- Add 3 text-only placeholder testimonials without real-person attribution.
- Add a Gallery grid with at least 6 optimized images, each with meaningful English alt text.
- Add a Contact section with placeholders for phone, WhatsApp, address (Coimbatore), business hours, and a short services note; no map embeds.
- Add basic on-page SEO: page title, meta description, and proper heading hierarchy (one H1, section H2s).
- Store all generated images as static assets under `frontend/public/assets/generated` and reference them via static paths.

**User-visible outcome:** Visitors can browse a polished single-page site, view services and gallery photos, read trust-focused content and testimonials, and contact the business via Call/WhatsApp from any device.
